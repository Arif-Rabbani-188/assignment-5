Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer:
1. The innerHTML property returns:
The text content of the element, including all spacing and inner HTML tags.
2. The innerText property returns:
Just the text content of the element and all its children, without CSS hidden text spacing and tags, except script tag and style tag elements.
3. The textContent property returns:
The text content of the element and all descendaces, with spacing and CSS hidden text, but without tags.

Question-3: What is event delegation in the DOM?

Answer:
Event delegation in the DOM is a JavaScript technique where you attach a single event listener to a parent element, allowing it to handle events triggered on its child elements, instead of adding separate listeners to each child element.

Question-4: What is event bubbling in the DOM?

Answer:
Event bubbling refers to the behavior where when an event occurs on an element, it triggers not only on that element but also propagates upwards through the DOM hierarchy, firing event handlers on its parent elements until it reaches the topmost element like the document or window object
